Types de données traitées

L’application ne collecte pas directement de noms, d’adresses e-mail, de numéros de téléphone ou d’informations de compte. Les services tiers utilisés peuvent traiter des données limitées telles que l’identifiant publicitaire, les identifiants d’appareil, les interactions avec l’application, les rapports de plantage et les informations d’achat/abonnement.

Finalités

Les données peuvent être utilisées pour la publicité, la gestion des abonnements, le fonctionnement de l’application, l’analyse, le dépannage, la sécurité et l’amélioration des performances.

Partage des données

Des données peuvent être partagées avec Google AdMob, Firebase Analytics, Firebase Crashlytics et RevenueCat. Le développeur ne collecte pas de données utilisateur via un serveur propriétaire distinct.

Sécurité des données

Les données sont traitées via les connexions sécurisées et les normes de sécurité des fournisseurs tiers concernés. Le contenu créé par l’utilisateur est stocké localement sur l’appareil.

Suppression des données

Comme l’application ne dispose pas de comptes utilisateur, il n’existe pas de système de suppression basé sur un compte. La désinstallation de l’application peut supprimer les données locales de l’appareil. Les données traitées par les services tiers sont soumises aux paramètres de confidentialité de ces fournisseurs.
