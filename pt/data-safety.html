Tipos de dados processados

O aplicativo não coleta diretamente nomes, e-mails, números de telefone ou informações de conta. Os serviços de terceiros usados podem processar dados limitados, como ID de publicidade, identificadores do dispositivo, interações no aplicativo, relatórios de falhas e informações de compra/assinatura.

Finalidades

Os dados podem ser usados para publicidade, gerenciamento de assinaturas, funcionalidade do aplicativo, análise, solução de problemas, segurança e melhoria de desempenho.

Compartilhamento de dados

Os dados podem ser compartilhados com Google AdMob, Firebase Analytics, Firebase Crashlytics e RevenueCat. O desenvolvedor não coleta dados do usuário por meio de um servidor proprietário separado.

Segurança dos dados

Os dados são processados por conexões seguras e pelos padrões de segurança dos respectivos provedores terceiros. O conteúdo criado pelo usuário é armazenado localmente no dispositivo.

Exclusão de dados

Como o aplicativo não possui contas de usuário, não há um sistema de exclusão baseado em conta. A desinstalação do aplicativo pode remover os dados locais do dispositivo. Dados processados por serviços de terceiros estão sujeitos aos controles de privacidade desses provedores.
