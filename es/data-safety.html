Tipos de datos procesados

La aplicación no recopila directamente nombres, correos electrónicos, números de teléfono ni información de cuenta. Los servicios de terceros utilizados pueden procesar datos limitados como ID de publicidad, identificadores del dispositivo, interacciones con la aplicación, informes de fallos e información de compra/suscripción.

Finalidades

Los datos pueden utilizarse para publicidad, gestión de suscripciones, funcionalidad de la aplicación, análisis, resolución de problemas, seguridad y mejora del rendimiento.

Compartición de datos

Los datos pueden compartirse con Google AdMob, Firebase Analytics, Firebase Crashlytics y RevenueCat. El desarrollador no recopila datos de usuario mediante un servidor propietario independiente.

Seguridad de los datos

Los datos se procesan mediante conexiones seguras y los estándares de seguridad de los proveedores terceros correspondientes. El contenido creado por el usuario se almacena localmente en el dispositivo.

Eliminación de datos

Como la aplicación no tiene cuentas de usuario, no existe un sistema de eliminación basado en cuentas. Desinstalar la aplicación puede eliminar los datos locales del dispositivo. Los datos procesados por servicios de terceros están sujetos a los controles de privacidad de dichos proveedores.
