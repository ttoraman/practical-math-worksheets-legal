Información general

Practical Math Worksheets es una aplicación móvil diseñada para crear hojas de ejercicios de matemáticas. La aplicación no crea cuentas de usuario y no solicita nombres, direcciones de correo electrónico, números de teléfono ni información personal similar.

Funcionamiento local

Las funciones principales de la aplicación se realizan localmente en el dispositivo. Las hojas y contenidos creados por el usuario no se envían a ningún servidor propiedad del desarrollador.

Servicios de terceros

La aplicación puede usar Google AdMob, Firebase Analytics, Firebase Crashlytics y RevenueCat. Estos servicios pueden procesar datos técnicos limitados para publicidad, análisis de uso, informes de fallos y gestión de suscripciones.

Publicidad

La aplicación puede mostrar anuncios mediante Google AdMob. AdMob puede procesar datos como el ID de publicidad, información del dispositivo y datos de uso para mostrar anuncios y medir su rendimiento.

Análisis e informes de fallos

Firebase Analytics puede utilizarse para comprender el uso de la aplicación, y Firebase Crashlytics para analizar fallos y errores. Esta información se utiliza para mejorar el rendimiento y la estabilidad.

Suscripciones y compras

La aplicación puede ofrecer opciones de suscripción para funciones premium. Las compras y suscripciones se gestionan a través de Google Play y RevenueCat. El desarrollador no accede ni almacena los datos de tarjeta de crédito o pago.

Compartición de datos

La aplicación no envía datos a servidores propios del desarrollador. Sin embargo, los servicios de terceros mencionados pueden procesar datos técnicos y de transacción de acuerdo con sus propias políticas de privacidad.

Privacidad de los niños

La aplicación no recopila conscientemente información personal de niños.
